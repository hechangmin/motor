***motor ？ what's this***
--------------

* 这是用nodejs写的一个web服务器框架，谈不上什么MVC模式。

不过其流程清晰、配置简单、操控性强，就犹如摩托车一样 sexy。*

----------------------------------------------------------------------

motor与express比较
--------------

>犹如他们的名字一样：
   *express 功能比较完善，封装比较复杂，喜欢他的人会很喜欢，比如说你。
   *motor 很简单，操控性强，基本功能应该还不耐，喜欢他的人会很喜欢，比如说我。
   *对此，我只能说萝卜青菜各有所爱。*
